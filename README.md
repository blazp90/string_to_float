Helper function for changing string to float value.

Example:
2.110,60       => 2110.6
2,110.60       => 2110.6
2.110.60       => 2110.6
2,110,60       => 2110.6
89,561,52.66   => 8956152.66
89.561.52,66   => 8956152.66
20.2,1,10.6085 => 202110.6085
2.A1b10.60     => 2110.6
2,A1b10.60     => 2110.6
2,A1b,10.6000  => 2110.6
2,A1b,10.6080  => 2110.608
